# 🎵 音乐格式转换器 - 快速开始指南

## 📋 项目概览

这是一个现代化的音乐格式转换工具，具有以下特点：

- ✅ **图形化界面** - 现代化深色主题，操作简单直观
- ✅ **批量转换** - 一键转换整个文件夹的音乐文件
- ✅ **单文件转换** - 支持单独转换指定音乐文件
- ✅ **多格式支持** - MP3, WAV, FLAC, AAC, OGG, M4A 等
- ✅ **实时进度** - 显示转换进度和详细状态
- ✅ **线程安全** - 后台转换，界面不卡顿

## 📦 环境要求

### 必需组件
- **Python 3.8+**（已通过uv管理）
- **ffmpeg**（系统级依赖，需手动安装）

### 安装ffmpeg（Windows）
1. 下载 ffmpeg：https://ffmpeg.org/download.html
2. 解压到 `C:\ffmpeg`
3. 添加到系统PATH：
   - 右键"此电脑" → 属性 → 高级系统设置
   - 环境变量 → 系统变量 → Path
   - 添加 `C:\ffmpeg\bin`

### 验证ffmpeg安装
```bash
ffmpeg -version
```

## 🎯 使用方法

### 1. 转换单个文件
1. 点击"选择音乐文件"按钮
2. 选择要转换的音乐文件
3. 选择输出格式（如MP3）
4. 点击"开始转换"

### 2. 批量转换文件夹
1. 点击"选择音乐文件夹"按钮
2. 选择包含音乐的文件夹
3. 选择输出格式
4. （可选）设置输出目录
5. 点击"开始转换"

### 3. 自定义输出目录
- 在"输出目录"字段输入路径，或点击"选择目录"
- 留空则使用默认输出目录

## ⚠️ 常见问题

### Q: 提示"找不到ffmpeg"
A: 确保ffmpeg已正确安装并添加到系统PATH

### Q: 程序无法启动
A: 检查是否已安装依赖：`uv pip install -r requirements.txt`

### Q: 转换失败
A: 检查文件是否损坏，或格式是否支持

### Q: 转换速度慢
A: 大文件转换需要时间，这是正常现象
